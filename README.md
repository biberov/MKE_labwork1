## 🎯 Цель работы
Разработка программного комплекса для численного интегрирования напряжений по толщине оболочечного конечного элемента с целью вычисления усилий `N` и моментов `M`. Реализация и сравнительный анализ методов:
- Составная формула трапеций
- Составная формула Симпсона
- Квадратуры Гаусса с 3, 5 и 7 узлами
